
用途：
使用netapp做内网穿透，将本地ip地址127.0.0.1:8088(tomcat服务)做成一个互联网可以访问的域名，双击运行exe文件即可。

示例
http://4vbpk5.natappfree.cc -> 127.0.0.1:8088
 
其中http://4vbpk5.natappfree.cc 是生成互联网可以访问的暂时性域名（每次双击生成的都不一样），127.0.0.1:8088,是这个域名代理的ip地址和端口，
该内网穿透只要是为了支付宝付款成功的回调通知用的，当在商城上扫描二维码付款成功之后，支付宝会回调通知，但是支付宝只能访问公网，无法访问我本地，因此使用内网穿透把127.0.0.1:8088 模拟成公网上的服务提供支付宝回调使用。
